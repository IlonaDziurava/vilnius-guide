# Useful Apps

Download these before you arrive.

---

| App                  | What it does                                      | Essential?                |
| -------------------- | ------------------------------------------------- | ------------------------- |
| **Bolt**             | Ride-hailing (like Uber, but dominant in Vilnius) | Yes                       |
| **Trafi**            | Public transport routes and real-time schedules   | Yes                       |
| **Google Maps**      | Navigation — works excellently in Vilnius         | Yes                       |
| **Google Translate** | Lithuanian menus and signs                        | Useful                    |
| **Bolt Food**        | Food delivery                                     | Optional                  |
| **M-Ticket**         | Cheaper bus/trolleybus tickets (€0.65 vs €1.00)   | If using public transport |

---

## Notes

**Bolt** is the app to use for taxis in Lithuania. It is significantly
cheaper than street taxis and you avoid any pricing disputes.
Always use the app, never hail a taxi from the street.

**Trafi** covers all Vilnius buses and trolleybuses with real-time
tracking. You can also buy tickets through the app.

**M-Ticket** — if you plan to use public transport regularly,
download this and load it with credit. Tickets cost €0.65 instead
of €1.00 cash, and a day pass is €3.00.
