# Phrasebook

A quick-reference card of Lithuanian phrases for your trip.

---

## Greetings

| Lithuanian    | English        |
| ------------- | -------------- |
| Labas         | Hello          |
| Labas rytas   | Good morning   |
| Labas vakaras | Good evening   |
| Viso gero     | Goodbye        |
| Iki           | Bye (informal) |

| Lithuanian | English                 |
| ---------- | ----------------------- |
| Taip       | Yes                     |
| Ne         | No                      |
| Prašom     | Please / You're welcome |
| Ačiū       | Thank you               |
| Labai ačiū | Thank you very much     |
| Atsiprašau | Excuse me / Sorry       |
| Gerai      | OK / Good               |
