# Vilnius Travel Guide

A complete travel documentation site for Vilnius, Lithuania — built with MkDocs and the Material theme.

Live site: [https://ilonadziurava.github.io/vilnius-guide/](https://ilonadziurava.github.io/vilnius-guide/)

## Setup

```bash
pip install mkdocs-material
mkdocs serve
```

## Deploy

```bash
mkdocs gh-deploy
```
